# Somiya El Mimouni Accessibility Portfolio

A React portfolio website in English for digital accessibility audit work.

## Features

- English content
- Page language set with `lang="en"`
- Semantic page regions and heading structure
- Visible black focus indicator
- Responsive layout
- Accessible navigation, buttons, and links

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
